# Automated Phase Reconstruction using ML Algorithms through Holography
__________________________________________________________________________

## Overview
This application implements a computational pipeline for the reconstruction and analysis of quantitative phase images. It enables systematic extraction of quantitative features and supports predictive modeling based on reconstructed phase information.

## Key Features
- Reconstruction of holographic microscopy images from object and reference inputs
- Automated feature extraction from reconstructed data
- Integrated analysis pipeline including segmentation and quantitative metrics
- Machine learning-based prediction using extracted features

## Workflow
1. Load an object image and corresponding reference images
2. Generate reconstructions for selected frequency peaks
3. Inspect and select the most suitable reconstruction
4. Run analysis to obtain segmentation and feature data
5. Generate predictions based on extracted features

## Notes
- The quality of reconstruction depends on the input hologram and selected frequency peaks  
- Multiple reconstructions can be compared to identify optimal results
